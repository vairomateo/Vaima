# Querandí Mates

Proyecto final desarrollado con JavaScript para simular una tienda online de mates, yerbas y accesorios.

## Tecnologías utilizadas

- HTML5
- CSS3
- SCSS
- JavaScript ES6
- Bootstrap 5
- SweetAlert2
- Fetch API
- LocalStorage

## Funcionalidades

- Renderizado dinámico de productos
- Carrito de compras
- Persistencia con LocalStorage
- Filtros por categoría
- Buscador dinámico
- Responsive Design
- Modularización de JavaScript
- Consumo de JSON mediante Fetch
